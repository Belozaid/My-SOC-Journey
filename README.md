# My-SOC-Journey
# رحلتي لتصبح محلل أمن سيبراني (SOC Analyst)

الخطوات الأولى نحو تخصص SOC Analyst

لا يهم مدى سرعة التقدم بقدر ما يهم الاستمرار وإتقان الأدوات. خلال هذه المرحلة، ركزت على بناء الأساس التقني وتجهيز البيئة العمليّة للبدء في تحليل التهديدات والشبكات.

🔍 أبرز ما تم إنجازه وتقنيات التعلم:

• تحليل الشبكات (Traffic Analysis):
• بدأت الغوص في أعماق البروتوكولات باستخدام أداة Wireshark.
• تمكنت من تصفية حزم الـ DNS وتحليل البيانات المستخرجة لفهم كيفية تواصل الأجهزة.

• التطبيق العملي (Hands-on Labs):
• انضممت لمنصة TryHackMe وبدأت بمهام "Advent of Cyber 2024" لتعزيز المهارات الدفاعية (Blue Teaming).

• التوثيق وبناء الهوية التقنية:
• تجهيز منصة GitHub الخاصة بي لتكون مرجعاً لكل المشاريع والسيناريوهات التي سأقوم بحلها مستقبلاً.

🚀 الأهداف القادمة (Phase 2):
في المرحلة المقبلة، سينصب التركيز على:
1. التعمق في فلاتر Wireshark المتقدمة.
2. تحليل بروتوكولات ICMP وفهم آلية عمل الـ Ping من منظور أمني.
3. الاستمرار في حل تحديات الأمن السيبراني لتعزيز مهارات التحقيق الرقمي.
"الرحلة مستمرة، والتعلم لا يتوقف عند جدول زمني محدد."
